

## 一、技术栈

○ 前端：HTML、CSS、JavaScript、Semantic UI、
○ 前端工具： HBuilderX

○ 后端：Spring Boot 2.2.5、Thymeleaf 3.0.9、 JPA、 MySQL 5.7、 JDK 1.8、 Maven 3.6.1
○ 后端工具： Intellij IDEA 2019.3

○ 脑图：有道云笔记

○ 插件：
- typo.css 排版
- prism 代码高亮
- Tocbot 生成目录
- animate.css 动画效果
- jquery.scrollTo 平滑滚动
- commonmark HTML转MarkDown

○ 服务器：阿里云 EMS
○ 图片存储： 腾讯云 COS
